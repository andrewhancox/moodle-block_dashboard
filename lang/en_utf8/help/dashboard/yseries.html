Dashboard element

Graphical data serie

Data series will be plotted on graph display. Depending on the graph type, you will be able to specify one or more data series for plotting. Data series MUST be output column names (or aliases) and are given as a semi-column (";") separated list.

Names of series can be defined in the textarea at the right. Labels will be separated by semi-columns (";").

Special features on data series

When a serie is mentionned as :

S(serie_name)

output values will be accumulated in order of the display.
