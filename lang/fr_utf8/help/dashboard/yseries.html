Séries de données

Les séries de données sont les données graphées. Suivant le type de graphe il est possible de définir une ou plusieurs séries à afficher. Les séries de données doivent êtres de noms de colonnes de sortie (ou leurs alias) et doivent être séparées par des ";".

Les noms des séries peuvent être définies dans la zone de texte à droite (Titre des séries) sous forme d'une liste de mentions séparées par des ";".

Fonctions spéciales sur les séries de sorties

Si une série de données est exprimée sous forme S(nom_colonne), alors les valeurs sont cumulées dans l'ordre d'affichage de la sortie.
