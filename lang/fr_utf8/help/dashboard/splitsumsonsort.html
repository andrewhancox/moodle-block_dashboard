Element de tableau de bord

Sous-totaux de données triées

Lorsque les données sont triées, il est possible d'obtenir des sous-totaux partiels sur un critère de séparation. L'ensemble des sommateurs déclarés sera traité et une ligne de sous-totaux apparaîtra à chaque changement de valeur.

Le critère de découpage est le nom d'une colonne de sortie. La table de sorite doit être triable et triée effectivement sur le critère de séparation pour que l'affichage se produise.

Seules les tables linéaires de données peuvent bénéficier de cette fonctionnalité
