Element de tableau de bord

Type de table de sortie de données

Ce sélecteur permet de choisir l'un des types de tables suivants :

  • Table linéaire : Les données sont présentées par ligne de résultat
    • Tableau croisé : Les données sont présentées sous forme d'un tableau à deux dimensions. La description de la table doit préciser:
      • La colonne portant la dimension horizontale (un seul choix)
      • La ou les colonnes portant la dimension verticale.
      • La composition de la cellule contenu
    • Table hiérachique : si les données extraites présentent un principe hiérachique (id,parent) il est possible de les afficher dans une représentation arborescente.
