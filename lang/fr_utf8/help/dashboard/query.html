Element de tableau de bord

Requête

Le tableau de bord appuie ses affichages sur une requête permettant d'obtenir les données source.

Cette requête :

  • doit présenter des champs de sortie nommés (si besoin par des clauses AS
  • peut présenter des jointures
  • peut utiliser des fonctions d'aggrégation et des clauses GROUP BY
  • ne doit pas présenter de clauses ORDER BY si les données sont destinées à un afficage en table de données

Introduction des filtres

Si des filtres doivent être définis sur les données, alors il est nécessaire d'insérer une balise <%%FILTERS%%> dans la requête à la place ou pour compléter une claise WHERE
